The GitHub Matrix Screensaver

The GitHub Matrix Screensaver for Mac OSX

Download as .zip Download as .tar.gz View on GitHub
The latest commits from GitHub visualized in a Matrix-style animation.

The GitHub Matrix Screensaver for Mac OSX shows a constant stream of recent commits from GitHub. It's based on my GitHub Matrix webapp project and on Tom Robinsons WebSaver project (kudos).

http://winterbe.com/projects/github-matrix/

For more infos about the GitHub Matrix see this repository:

https://github.com/winterbe/github-matrix

Install (Mac OSX only)

  • Fork or clone this project, then double click the file Web.saver in the project folder.
  • Open the Mac OSX preferences, choose Screensaver and activate Web

Uninstall

Open the Mac OSX preferences, choose Screensaver, right click on Web and choose delete.
